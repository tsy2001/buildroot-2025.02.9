# Put your selected font C files here

Place only the font `.c` files you want to ship, e.g.:

- u8g2_font_ncenB14_tr.c
- u8g2_font_misans_16_cjk.c

These files are typically generated by `bdfconv` (u8g2/tools/font/bdfconv).

The build will compile every `*.c` in this directory into libu8g2font.so.
